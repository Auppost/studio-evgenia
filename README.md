# Studio Evgenia — сайт мастера депиляции и электроэпиляции

Продакшн-реализация дизайна из хендоффа (`design_handoff_studio_evgenia`) на **React + Vite**.
Трёхъязычный (RU / ET / EN) одностраничный сайт студии в Таллинне: депиляция, электроэпиляция
(главный оффер — стойкое удаление навсегда), массаж. Главное целевое действие — онлайн-запись,
заявки уходят в Instagram Direct.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
npm run preview  # предпросмотр собранного билда
```

Требуется Node 18+ (проверено на Node 22).

## Структура

```
index.html              — базовые meta-теги, шрифты Google (Cormorant Garamond + Jost)
src/
  main.jsx              — точка входа
  App.jsx               — оркестратор: state (язык, страница, фильтр, запись), per-language SEO
  styles.css            — все дизайн-токены и стили (цвета, типографика, отступы, адаптив)
  i18n.js               — словарь TR (ru/et/en), финальный копирайтинг
  data.js               — SERVICES, REVIEWS, GALLERY, FEATURED, контакты
  helpers.js            — локализация услуг, генерация дней для записи
  components/
    Header.jsx          — topbar + sticky-шапка, навигация, переключатель языка
    Footer.jsx
    Home.jsx            — hero, «обо мне», аудитории, процесс, услуги-тизер, отзывы, FAQ, CTA
    Services.jsx        — список услуг по категориям + фильтр аудитории
    Gallery.jsx         — masonry-сетка фото
    Reviews.jsx
    Contacts.jsx        — контакты + демо-форма обратной связи
    Booking.jsx         — мастер записи (услуга → дата → время → подтверждение)
public/uploads/         — фотографии студии
```

## Что нужно заменить на реальные данные

- **Телефон / WhatsApp:** `+372 5xxx xxxx` — в `src/i18n.js` (ключ `c_phone_v` во всех трёх языках).
- **Адрес:** сейчас «точный адрес при записи» (`c_address_v`).
- **Instagram:** ссылка и хэндл — в `src/data.js` (`IG_URL`, `IG_HANDLE`).
- **Цены:** массив `SERVICES` в `src/data.js` (сейчас ориентировочные).
- **Фото:** файлы в `public/uploads/`.
- **Таблица записей для мастера** — запись отправляется в Google Таблицу через Apps Script.
  Мастер открывает таблицу и видит все записи строками. Настройка (≈5 мин) — см.
  [`SETUP-BOOKING.md`](SETUP-BOOKING.md) и скрипт [`booking-sheet.gs`](booking-sheet.gs).
  Пока `BOOKING_ENDPOINT` в `src/data.js` пустой, кнопка записи откатывается на Instagram Direct.
- **Форма обратной связи** (`Contacts.jsx`) — пока демо: показывает благодарность, ничего не
  отправляет. При желании можно подключить к тому же Apps Script по аналогии с записью.

## Заметки по реализации

- **Навигация клиентская** (без перезагрузки), смена «страницы» через state, анимация `fadeUp`.
- **Язык** — глобальный state в `App.jsx`; при смене обновляются `<html lang>`, `<title>` и
  `meta description` (базовая per-language SEO). Для полноценного SEO с pre-render можно
  перенести на Next.js (SSG + hreflang) — данные и словарь уже вынесены и переносятся как есть.
- **Вариант hero** переключается константой `HERO_VARIANT` в `App.jsx` (`'A'` фон-фото / `'B'` сплит).
- В сводке записи услуга и дата фиксируются на языке момента выбора (как в референсе).
