# Таблица записей для мастера — настройка

Записи с сайта попадают в **Google Таблицу**. Мастер открывает таблицу и видит все записи
строками: когда получено, услуга, день, время, имя, контакт, язык сайта, статус.

## Шаги (≈5 минут)

1. **Создайте таблицу** — откройте [sheets.new](https://sheets.new), назовите её, например
   «Записи — Studio Evgenia».
2. **Откройте редактор скрипта** — в таблице меню **Расширения → Apps Script**.
3. **Вставьте код** — удалите содержимое `Code.gs` и вставьте целиком файл
   [`booking-sheet.gs`](booking-sheet.gs) из этого проекта. Сохраните (Ctrl/Cmd+S).
   - *(необязательно)* впишите свой email в `NOTIFY_EMAIL` — будете получать письмо о каждой записи.
   - *(необязательно)* заполните `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` — см. раздел
     **«Уведомления в Telegram»** ниже — и записи будут мгновенно приходить в Telegram.
4. **Разверните веб-приложение** — кнопка **Развернуть → Новое развёртывание**:
   - Тип: **Веб-приложение**
   - Запуск от имени: **Я**
   - У кого есть доступ: **Все (Anyone)**
   - Нажмите **Развернуть**, при первом разе подтвердите доступ к аккаунту.
   - Скопируйте **URL веб-приложения** (заканчивается на `/exec`).
5. **Впишите URL в сайт** — откройте `src/data.js`, вставьте URL в кавычки:
   ```js
   export const BOOKING_ENDPOINT = 'https://script.google.com/macros/s/…/exec'
   ```
6. Пересоберите/перезапустите сайт (`npm run dev` или `npm run build`). Готово.

## Как это работает

- Клиент проходит запись (услуга → день → время → имя + контакт) и жмёт **«Отправить запись»**.
- Данные уходят на веб-приложение Apps Script, которое дописывает строку в таблицу.
- Заголовок таблицы создаётся автоматически при первой записи; первая строка закрепляется.
- Колонки: **Получено · Дата · Время · Услуга · Имя · Контакт · Язык · Статус**.

### Занятость слотов (защита от двойной записи)

- Сайт спрашивает у таблицы, какие слоты уже заняты, и показывает их как **«занято»** —
  выбрать это время нельзя.
- Слот = **день + час** (мастер один, одна запись в час).
- При отправке скрипт под блокировкой (LockService) проверяет, что слот ещё свободен. Если
  за эти секунды его заняли — бронь отклоняется, клиент видит «это время только что заняли»
  и выбирает другое. **Два человека не смогут записаться на один слот.**
- **Отмена освобождает слот:** поставьте в колонке «Статус» значение **`Отменена`** —
  и это время снова станет доступным на сайте.
- Колонку «Статус» мастер ведёт вручную: `Новая` → `Подтверждена` / `Отменена`.

## Пока URL не вписан

Если `BOOKING_ENDPOINT` пустой, кнопка на шаге подтверждения открывает **Instagram Direct**
(старое поведение). Сайт полностью рабочий — таблица подключается, когда будете готовы.

## Проверка

Открыв URL веб-приложения в браузере, вы должны увидеть
`{"ok":true,"service":"studio-evgenia booking endpoint"}` — значит эндпоинт живой.
После тестовой записи с сайта в таблице появится новая строка.

## Уведомления в Telegram (необязательно)

Чтобы получать сообщение в Telegram о каждой новой записи:

1. **Создайте бота** — в Telegram напишите [@BotFather](https://t.me/BotFather), команда `/newbot`,
   задайте имя и username. BotFather пришлёт **токен** вида `123456789:AAE...` — это `TELEGRAM_BOT_TOKEN`.
2. **Узнайте свой chat id** — напишите [@userinfobot](https://t.me/userinfobot), он ответит вашим
   числовым `Id` (например `483920114`) — это `TELEGRAM_CHAT_ID`.
3. **Нажмите «Start»** в чате со своим новым ботом (иначе бот не сможет вам писать).
4. **Впишите оба значения** в `booking-sheet.gs` (переменные `TELEGRAM_BOT_TOKEN` и
   `TELEGRAM_CHAT_ID`), сохраните и переразверните скрипт (см. «Обновление скрипта» ниже).

После этого при каждой записи в Telegram придёт сообщение вида:

```
🗓 Новая запись

Дата: 2026-07-15
Время: 14:00
Услуга: Голени
Имя: Анна
Контакт: @anna
```

**Групповой чат вместо личного:** добавьте бота в группу, в `TELEGRAM_CHAT_ID` укажите id группы
(отрицательное число). Узнать id группы можно, добавив в неё @userinfobot, либо через
`https://api.telegram.org/bot<ТОКЕН>/getUpdates` после сообщения в группе.

## Обновление скрипта

Если позже поменяете `booking-sheet.gs`, в Apps Script нужно сделать
**Развернуть → Управление развёртываниями → (карандаш) → Новая версия**, иначе изменения
не применятся к тому же URL.
